# grep_tool

- Use for locating functions, props, routes, errors, or config keys.
- Search for the smallest distinctive string that can anchor the investigation.
- Prefer exact identifiers, error text, route names, and config keys over broad natural-language terms.
- Use follow-up searches to narrow from file discovery to the exact block that needs reading or editing.
- In huge files, search for nearby function names, comments, JSX labels, or schema keys to jump to the right region.
